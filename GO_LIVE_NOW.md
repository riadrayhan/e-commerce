# GO LIVE IN 5 MINUTES ⚡

## Your Website is Ready - No More Localhost!

### Step 1: Click Publish (1 minute)
In v0 editor, top right → Click **"Publish"** button → Wait for deployment

### Step 2: Get Your Live URL
After publish completes, you'll see your live website URL like:
```
https://v0-e-commerce-website-build.vercel.app
```

### Step 3: Set Environment Variables (2 minutes)
Go to https://vercel.com/dashboard:

1. Click your project
2. Settings → Environment Variables
3. Add 3 variables:
   - `DATABASE_URL` = your Neon connection string
   - `ADMIN_PASSWORD` = any password (example: `Store@123`)
   - `JWT_SECRET` = any random string (example: `abc123xyz`)

4. Redeploy (click Deployments → Redeploy)

### Step 4: Done! 🎉
Your live website is at: `https://your-project-name.vercel.app`

## Design Features

### Color Usage

**Blue (#5B7CE6)**
- Primary buttons
- Links and CTAs
- Section highlights
- Hover states
- Icons in features

**Orange (#E8652F)**
- Stock status badges
- Secondary accents
- Attention-grabbing alerts
- Secondary buttons

**Neutral Colors**
- White backgrounds
- Gray text for secondary info
- Light gray for muted elements
- Dark gray for body text

### Typography

- **Large Headlines**: Bold, 48-56px
- **Section Titles**: Bold, 32-40px
- **Card Titles**: Semibold, 20-24px
- **Body Text**: Regular, 16px
- **Small Text**: Regular, 14px

### Spacing

- Consistent 16px base spacing
- Cards have 24px padding
- Sections separated by 48-64px
- Margin-bottom between elements

### Interactions

- **Hover**: Shadow increase, color change
- **Click**: Visual feedback, instant
- **Loading**: Smooth skeleton screens
- **Transitions**: 200-300ms animations
- **Mobile**: Touch-friendly, larger targets

### Images

- Product images: 192px × 192px
- Hero images: Full width
- Proper aspect ratios (object-cover)
- Fallback icons when no image
- Base64 encoding for performance

## Responsive Design

### Mobile (< 640px)
- Single column products
- Full-width cards
- Stacked buttons
- Reduced padding
- Optimized font sizes
- Touch-friendly buttons (44px minimum)

### Tablet (641-1024px)
- 2-3 column grid
- Balanced spacing
- Multi-line buttons
- Adjusted padding

### Desktop (1025px+)
- 4-column product grid
- Full feature set
- Maximum 1280px width
- Rich hover effects
- Multi-column layouts

## Accessibility Features
